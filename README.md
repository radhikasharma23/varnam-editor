# Varnam Editor

Browser & Desktop compatible editor frontend for [Varnam](https://TypeIndianLang.Online).

## Project Setup

Install dependencies :
```
npm install
```
Compiles and hot-reloads for development :
```
npm run serve
```
Compiles and minifies for production :
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
